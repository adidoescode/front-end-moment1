Startsida Bilder Arbetsprocess

Den här uppgiften går ut på att få en inblick på hur NodeJS kan hjälpa till att automatisera publicering genom versionshanterings-program som t.ex github. Med hjälp av netlify kommer mina commits direkt uppdatera hemsidan då jag angett kommandot "npm run build" på netlify. NodeJS kommer också vara till användning för transpilering och minifiering av CSS, HTML och JavaScript samt automatiskt skapa optimerade grafikfiler.
