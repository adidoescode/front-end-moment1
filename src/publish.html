Startsida Bilder Arbetsprocess

Först skapade jag en mapp vid namn moment1. Sedan navigerade jag till mappen och skrev i kommandotolken "npm init". Där skrev jag in author och lämnade resten tomt. Jag behövde parcel för denna uppgiften, så jag importerade paketet genom att skriva "npm install parcel". Detta gav mig en mapp vid namn "node_modules" och ".parcel-cache". Sedan ändrade jag scripts i package.json-filen till detta:

"scripts": {
"start": "parcel",
"build": "parcel build"
}

"start": "parcel" gör så att vi kan starta en localhost som visar uppdateringar i realtid utan användning av Live-server. Det andra kommandot skapar en dist mapp som innehåller de transpilerade filerna. Dessa filer är de som kommer publiceras på hemsidan när vi uppdaterar github-repot.

Genom att koppla github-repot till netlify och ge den build command:

npm run build

kommer sidan automatiskt uppdateras när jag pushar nya ändringar till repot. Publish directoryn är satt till dist då det är de distruberade filerna som ska laddas upp.
